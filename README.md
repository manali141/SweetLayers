# 🎂 Sweet Layers Cake Shop

Cake shop website built using **HTML, CSS, and JavaScript**.  
This is a **frontend-only project** designed with modern UI, flexbox layouts, and interactive features.

---

## 🌐 Live Features

### 🏠 Home Page
- Hero section with bakery-themed design
- Featured cake cards in a responsive flex layout
- About section describing the shop

### 🍰 Menu Page
- Displays all cakes in a **card grid layout**
- Real-time **search functionality**
- Each cake card includes:
  - Image
  - Name
  - Category badge
  - Description
  - Price
  - Order button

### 📞 Contact Page
- Fully styled contact form with validation
- Fields include:
  - Name
  - Email
  - Phone
  - Cake Type
  - Event Date
  - Message
- Bakery contact details:
  - Address
  - Phone
  - Email
  - Working hours

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox & Responsive Design)
- JavaScript (DOM Manipulation & Search Filter)
